API DOCUMENTATION

BASE URL

api-jollibee-menu.vercel.app

ENDPOINTS

GET /menu
returns a list of all available menu items.
GET /menu/:item
returns information about a specific menu item.

URL Parameters

item - the name of the menu item to retrieve.
